# Rules: 모델 선택 / 컨텍스트 (범용 프로세스 정책)

## 모델 선택

- **Opus = 오케스트레이터(메인)**: 전체 맥락 보유, A~F 진행·검토 대응, G에서 워크트리/위임/공유문서 관리·취합. 판단·통합 책임.
- **Sonnet = 서브에이전트(실행자)**: G의 API 1개 구현 같은 **국소적·반복적 작업**. 격리 컨텍스트.
- (Haiku는 현재 미사용 — 필요 시 단순 분류/포맷 작업에 한정 검토)

## 컨텍스트 관리

- 서브에이전트는 **격리 컨텍스트** → 매번 전체 맥락이 없으므로 **필요한 Rules/Skills를 명시 주입**받아 일관성 확보.
- 메인(Opus) 컨텍스트는 린하게: 불필요한 MCP/도구 비활성, 대용량 출력은 서브에이전트로 위임해 메인 오염 방지.
- 병렬 4개는 컨텍스트/리소스 한도 고려한 디폴트. 늘리려면 사람 승인.
